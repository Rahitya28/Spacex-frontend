# SpaceX Mission Tracker

## Overview

SpaceX Mission Tracker is a full-stack MERN web application that allows users to browse and monitor SpaceX missions through a modern and responsive dashboard. Users can search missions, filter them by status, sort them by launch date, and view detailed mission information.

The application is built using React for the frontend, Node.js and Express for the backend, and MongoDB Atlas for data storage.

## Features

- Modern SpaceX-inspired user interface
- Responsive design for desktop and mobile
- Search missions by name
- Filter missions by status
  - All Missions
  - Success
  - Scheduled
  - Failed
- Sort missions by launch date
  - Newest First
  - Oldest First
- View mission details
- Dynamic data fetched from MongoDB
- REST API integration
- Fast frontend powered by Vite
